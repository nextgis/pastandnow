# Vue2 NextGIS Web map component for leaflet

Plugin to integrate NextGIS Web maps into vue2.js with leaflet adapter

```html
<vue-ngw-map
  fullFilling
  baseUrl="https://demo.nextgis.com"
  webmapId="4251"
  qmsId="487"
></vue-ngw-map>
```
